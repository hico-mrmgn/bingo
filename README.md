# bingo

パーティー・イベント用のビンゴ番号抽選マシン。スクリーンに投影して使う想定。
ビンゴで当たった人向けの景品クジ引きアプリ（`lottery/`）も同梱。

## 使い方

`index.html` をブラウザで開くだけで動きます（ビルド・サーバー不要）。

- **抽選する** ボタン（またはスペースキー）で 1〜75 の番号を重複なく抽選
- 抽選済み番号は B / I / N / G / O の行ごとのボードに点灯表示
- 直近 5 個の番号をチップで表示（最新はハイライト）
- 抽選履歴は localStorage に保存されるため、リロードしても消えません
- **リセット** ボタンで履歴を消去（確認ダイアログ付き）

## 景品クジ（lottery/）

ビンゴで当たった人に引いてもらう景品クジ。`lottery/index.html` をブラウザで開くだけで動きます。

- **クジを引く** ボタン（またはスペースキー）で景品番号（デフォルト 1〜75）を重複なく抽選
- ドラムロール演出のあとファンファーレで番号を発表
- **本数を編集** ボタンでクジの本数を変更可能（1〜999。景品の数に合わせて調整）
- 抽選済み番号はボードに点灯表示、直近 5 個はチップで表示
- 本数・当選履歴は localStorage に保存されるため、リロードしても消えません
- **リセット** ボタンで履歴を消去（確認ダイアログ付き）

## 構成

| ファイル | 内容 |
|---|---|
| `index.html` | 画面構造 |
| `style.css` | スタイル（ダーク背景・投影向けの大きな文字） |
| `app.js` | 抽選ロジック・演出・状態保存 |
| `lottery/` | 景品クジ引きアプリ（構成は上と同じ 3 ファイル） |
