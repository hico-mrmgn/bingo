# bingo

パーティー・イベント用のビンゴ番号抽選マシン。スクリーンに投影して使う想定。

## 使い方

`index.html` をブラウザで開くだけで動きます（ビルド・サーバー不要）。

- **抽選する** ボタン（またはスペースキー）で 1〜75 の番号を重複なく抽選
- 抽選済み番号は B / I / N / G / O の行ごとのボードに点灯表示
- 直近 5 個の番号をチップで表示（最新はハイライト）
- 抽選履歴は localStorage に保存されるため、リロードしても消えません
- **リセット** ボタンで履歴を消去（確認ダイアログ付き）

## 構成

| ファイル | 内容 |
|---|---|
| `index.html` | 画面構造 |
| `style.css` | スタイル（ダーク背景・投影向けの大きな文字） |
| `app.js` | 抽選ロジック・演出・状態保存 |
